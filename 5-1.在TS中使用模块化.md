# 模块化

相关配置

|       配置名称        |               含义              |
|   :--------------:   |  :------------------------:     |
|       module         |    设置编译结果中的模块化标准      |
|   moduleResolution   |        设置解析模块的模式         |
| noImplicitUseStrict  |    编译结果中不包括“user strict”  |
|   removeComments     |        编译结果中移除注释         |
|   esModuleInterop    |    启用es模块化交互非es模块导出    |
> 前端领域中的模块化标准：ES6、commonjs、amd、umd、system、exnext
