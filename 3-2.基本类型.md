# 基本类型约束

> TS是一个可选的类型系统

# 如何进行类型约束

仅需要在 变量、函数的参数、函数的返回值位置加上```:类型```

ts 在很多场景中可以完成类型推导

any：表示任意类型，对该类型，ts不进行类型检查


# 源代码和编译结果的差异

编译结果中没有类型约束的信息

# 基本类型

- number:数字
- string:字符串
- boolean:布尔
- 数组
- object:对象
- null 和 undefined

null 和 undefined是所有其他类型的子类型，他们可以赋值给其他类型

通过添加```"strictNullChecks": true```，可以获得更加严格的空类型检查，null和undefined只能赋值给自身。
